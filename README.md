# 简介

IoT Studio是针对LiteOS操作系统的嵌入式开发环境，提供了编译、烧录及调试等一站式开发体验，支持 C、 C++、汇编等多种开发语言，让您快速，高效地进行物联网开发。

# 功能特性
## 快速开发
* 集成10分钟快速上云工程
* 兼容多种硬件平台
* 支持ARM Cortex-M、Cortex-A、RISC-V等多种内核芯片
* 快速创建端云协同的工程原型
## 快速定位
* 图形化调试定位
* 汇编调试
* 通信模组检测工具
## 图形化配置
* 智能一键化配置
* kconfig可视化配置管理
